## 1. 정의 및 적용 범위

### 1.1 주요 정의

- **개인정보**: 본 정책에 명시된 사용자 이름, 이메일 주소, 비밀번호 암호화 값, 가입 시간, 아바타 URL, 운영 로그 기록 등을 포함하여, 단독으로 또는 다른 정보와 조합하여 특정 개인을 식별할 수 있는 전자적 또는 기타 방식으로 기록된 정보를 의미합니다.
- **민감한 개인정보**: 유출되거나 불법적으로 사용될 경우 개인의 존엄성을 침해하거나 개인 또는 재산의 안전을 위협할 수 있는 개인정보를 의미합니다. 본 스튜디오는 어떠한 민감한 개인정보도 **수집하거나 처리하지 않습니다**.
- **타사 이미지 호스팅:** 사용자가 이 스튜디오와는 별개로 직접 선택하고 이미지를 업로드하는 공개 이미지 호스팅 서비스 플랫폼(예: SM.MS, Imgur, GitHub 등)을 의미합니다. 이 스튜디오는 이러한 플랫폼에서 반환되는 URL 문자열만 수신합니다.
- **Cloudflare:** Cloudflare, Inc.에서 제공하는 글로벌 CDN, DDoS 공격 방지, HTTPS 암호화 전송 및 R2 객체 스토리지 서비스를 의미합니다. Cloudflare는 이 서비스의 기술 인프라 제공업체이며, **독립적인 데이터 관리자**로서 서비스 제공에 필요한 네트워크 상호 작용 정보를 수집합니다.
- **Microsoft 계정 로그인:** Microsoft OAuth 2.0 프로토콜을 통해 구현된 타사 ID 인증 서비스를 의미하며, 사용자가 Microsoft 계정에 로그인하거나 이 웹사이트에 연결할 수 있도록 합니다.
- **OAuth 2.0 권한 부여 서비스:** 이 웹사이트에서 제공하는 OAuth 2.0 권한 부여 서비스를 의미하며, 사용자가 타사 애플리케이션에 웹사이트 계정 사용 권한을 부여할 수 있도록 합니다.
- **QR 코드 로그인:** QR 코드 스캔을 통해 구현되는 기기 간 신원 인증 서비스로, 사용자가 QR 코드를 스캔하여 모바일 기기에서 컴퓨터에 빠르게 로그인할 수 있도록 합니다.

## 2. 개인정보 수집 시나리오 및 목적

당사는 개인정보 보호법에 명시된 "최소한의 필요성" 원칙을 엄격히 준수하여 사용자 계정의 핵심 기능을 구현하는 데 필요한 범위 내에서만 정보를 수집합니다. 모든 정보 수집은 사용자의 **명시적 동의** 또는 **계약 이행에 필요한 경우**에 한합니다.

### 2.1 계정 등록 정보 수집

**수집 내용**:** 사용자가 등록 양식에 직접 입력한 **사용자 이름**, **이메일 주소**, **비밀번호**(원본 비밀번호는 클라이언트 측에서 HTTPS 암호화 채널을 통해 전송되며, 서버에는 복원 불가능한 Argon2id 알고리즘으로 암호화된 해시 값만 저장됩니다), **등록 시간**(서버 타임스탬프는 초 단위까지 정확함)

**수집 방법**: 사용자는 등록 페이지의 양식을 통해 정보를 제출합니다. 모든 항목은 필수 입력 사항이며, 어느 항목이라도 입력하지 않으면 등록이 완료되지 않습니다.

**수집 목적**: 고유한 사용자 식별자를 생성하고 계정 로그인, 신원 인증, 서비스 알림 등의 기본 기능을 제공하기 위함입니다.

**법적 근거**: 개인정보보호법 제13조 제1항 - 개인의 **명시적 동의** 획득

**보관 기간**: 계정 생성일부터 계정 삭제일까지 계정 삭제 시, 위 정보는 메인 데이터베이스에서 즉시 삭제됩니다. 백업 데이터는 백업 주기 동안 30일 이내에 자동으로 삭제되며, 삭제 후에는 복구할 수 없습니다.

### 2.2 아바타 표시 기능

본 웹사이트는 두 가지 아바타 소스를 지원하며, 처리 방식이 다릅니다.

#### 2.2.1 사용자 지정 아바타 (타사 이미지 호스팅 URL)

**수집 내용:** **타사 이미지 호스팅 서비스에서 사용자가 가져온 아바타 URL 문자열**입니다. 이 유형의 아바타의 경우, 저희 스튜디오는 원본 이미지 파일을 **수신, 저장 또는 처리하지 않습니다**.

**처리 과정:**
1. 사용자는 프로필 사진을 선택한 타사 이미지 호스팅 플랫폼에 업로드하고 공개된 URL을 가져옵니다.
2. 사용자는 해당 URL을 복사하여 본 웹사이트의 계정 설정 페이지에 있는 프로필 사진 필드에 붙여넣습니다.
3. 본 웹사이트는 URL을 텍스트 문자열로만 저장하고, 사용자의 프로필 페이지에 &lt;img&gt; 태그를 사용하여 표시합니다.
4. 이미지의 실제 전송, 로딩 및 캐싱은 사용자의 브라우저와 제3자 이미지 호스팅 플랫폼 간에 직접 처리됩니다. 저희 스튜디오 서버는 이미지 데이터 전송 과정의 어떤 부분에도 관여하지 않습니다.

**법적 고지**: 사용자가 제3자 이미지 호스팅 플랫폼에 이미지를 업로드하는 행위는 **사용자와 이미지 호스팅 서비스 제공업체 간의 별도 법적 관계**를 형성합니다. 저희 스튜디오는 사용자가 직접 제공한 URL을 단순히 수신하는 역할만 합니다. 사용자는 이미지 사용에 대한 법적 권리를 확보하고 이미지 콘텐츠가 타인의 권리를 침해하지 않도록 할 책임이 있습니다. 이미지 콘텐츠와 관련하여 발생하는 모든 법적 분쟁은 사용자와 이미지 호스팅 서비스 제공업체가 해결해야 하며, 저희 스튜디오는 이러한 분쟁을 검토할 의무가 없습니다.

#### 2.2.2 Microsoft 계정 아바타

**수집 콘텐츠**: Microsoft 계정에 로그인하거나 연결할 때, Microsoft로부터 아바타 이미지 데이터(base64 형식)를 가져옵니다.

**처리 과정**:
1. Microsoft OAuth 인증이 성공적으로 완료되면, Microsoft Graph API에서 아바타 이미지(base64 인코딩)를 가져옵니다.
2. 저장 공간 및 로딩 성능 최적화를 위해 base64 이미지 데이터를 **WebP 형식**으로 변환합니다.
3. 변환된 이미지 파일을 **Cloudflare R2** 객체 스토리지 서비스에 업로드합니다.
4. 아바타 표시를 위해 해당 이미지의 접근 URL을 데이터베이스에 저장합니다.

**저장 관련 참고 사항**: Microsoft 아바타의 실제 이미지 파일은 Cloudflare R2에 저장됩니다. 이는 사용자 정의 아바타(URL만 저장)와 달리, 당사가 직접 처리하고 저장하는 이미지 데이터입니다.

**법적 근거**: 개인정보보호법 제13조 제1항 - 개인의 **명시적 동의** 획득(Microsoft OAuth 인증 페이지에서 사용자가 동의한 내용).

**보관 기간**: 계정 유지 기간과 연동됩니다. 계정이 삭제되면 Cloudflare R2에 저장된 아바타 파일과 데이터베이스의 URL 기록이 동시에 삭제됩니다.

### 2.3 QR 코드 로그인 기능

**수집 내용**: QR 코드를 사용하여 로그인할 때, 기기 식별 및 로그인 보안 유지를 위해 **클라이언트 IP 주소**, **브라우저 사용자 에이전트 정보**, **QR 코드 세션 토큰**을 일시적으로 수집합니다.

**수집 방법**: QR 코드 생성 및 스캔 시 자동으로 수집됩니다. 모든 정보는 암호화되어 저장됩니다. 세션 토큰은 3분 동안만 유효합니다.

**수집 목적**: 컴퓨터에서 모바일 기기로 QR 코드를 스캔하여 간편하게 로그인할 수 있도록 기기 간 신원 인증을 지원합니다.

**법적 근거**: 개인정보보호법 제13조 제2항 - 개인이 당사자인 계약의 체결 및 이행에 필요함.

**보관 기간**: QR 코드 로그인 세션 토큰은 사용 후 즉시 삭제됩니다. 사용하지 않은 토큰은 자동으로 만료되어 3분 후 삭제됩니다.

### 2.4 OAuth 2.0 인증 서비스

**수집 내용**: 타사 애플리케이션에 웹사이트 계정 사용을 승인하는 경우, **타사 애플리케이션의 기본 정보**(애플리케이션 이름, 설명, 콜백 URL) 및 **승인 범위 기록**을 수집 및 저장합니다.

**수집 방법**: 승인 페이지에서 명시적으로 승인하는 행위를 통해 수집됩니다.

**수집 목적**: 신뢰할 수 있는 타사 애플리케이션에 웹사이트 계정 사용을 안전하게 승인하고, 관리 목적으로 승인 내역을 기록하기 위함입니다.

**법적 근거**: 개인정보보호법 제13조 제1항 - 개인의 **명시적 동의** 획득

**보관 기간**: 사용자가 직접 동의를 철회하거나 계정을 삭제할 때까지 동의 기록이 보관됩니다.

### 2.5 운영 로그 기록

**수집 내용**: 다음과 같은 사용자 운영 로그가 자동으로 기록됩니다.
- **계정 관리 작업**: 회원가입, 비밀번호 변경, 사용자 이름 변경, 아바타 변경, 계정 삭제
- **타사 계정 연동**: Microsoft 계정 연동/연동 해제
- **OAuth 인증 작업**: 타사 애플리케이션 인증, 인증 철회
- **계정 상태 변경**: 차단/차단 해제 (관리자 작업)

각 로그 항목에는 **사용자 UID**, **작업 유형**, **작업 시간**, **작업 상세 정보**가 포함됩니다.

**작업 세부 정보 설명**: 사용자 이름이나 아바타 변경과 같은 정보 변경 작업의 경우, 변경 전후의 값이 로그에 기록됩니다. 예를 들어, 사용자 이름을 변경하면 이전 사용자 이름과 새 사용자 이름이 기록되고, 아바타를 변경하면 이전 아바타 URL과 새 아바타 URL이 기록됩니다. 이러한 기록 데이터는 사용자의 개인 정보에 해당하며, 로그와 함께 6개월 동안 보관된 후 자동으로 삭제됩니다.

**수집 방법**: 시스템이 사용자가 이러한 작업을 수행할 때 자동으로 기록하며, 사용자의 추가 조치는 필요하지 않습니다.

**수집 목적**:
1. **보안 감사**: 비정상적인 운영 행위를 감지하고 계정 보안을 유지합니다.
2. **문제 해결**: 사용자의 계정 관련 문제를 해결합니다.
3. **법규 준수**: 운영 로그 보존에 대한 법적 및 규제적 요구 사항을 충족합니다.

**법적 근거**: 개인정보보호법 제13조 제2항 - 개인이 당사자인 계약의 체결 및 이행에 필요한 경우

**보관 기간**: 운영 로그는 **6개월** 동안 보관 후 자동 삭제됩니다. **보안 감사 목적으로 계정 해지 후 6개월 동안 운영 로그를 보관한 후 자동 삭제합니다.** 이 보관 기간은 계정 해지 시 즉시 삭제되는 계정의 핵심 데이터(사용자 이름, 이메일 등)에는 영향을 미치지 않습니다.

## 3. 데이터 이용에 대한 구체적인 설명

### 3.1 핵심 목적 범위

상기 수집된 개인정보는 **다음 목적에 한하여** **엄격히 제한적으로** 사용되며, 본 정책에 명시적으로 규정되지 않은 다른 목적으로는 **사용되지 않습니다**.
1. **본인 인증 및 계정 로그인**: 이메일과 비밀번호 해시를 이용하여 사용자 본인임을 확인하고, 비밀번호, Microsoft 계정, QR 코드 로그인 등 다양한 로그인 방식을 지원하여 계정에 대한 접근이 본인만 가능하도록 보장합니다.
2. **계정 관리**: 서비스 안정성 지표로 등록 기간을 기준으로 계정 사용 기간을 계산합니다.
3. **아바타 표시 서비스**: 사용자가 제공한 아바타를 프로필 페이지 또는 기타 영역에 표시하여 커뮤니티 인지도를 높입니다.
4. **서비스 알림**: 계정 보안에 필요한 알림(예: 인증 코드)을 이메일로 발송합니다. **마케팅 또는 홍보 이메일은 발송하지 않습니다.**
5. **OAuth 권한 관리**: 계정 보안을 위해 제3자 애플리케이션에 대한 권한을 관리합니다.
6. **보안 감사**: 운영 로그 기록 및 분석을 통해 계정 보안을 강화하고 비정상적인 활동을 감지합니다.
7. **법률 준수**: 법률 및 규정 또는 사법 절차에 따라 요구되는 경우 필요한 계정 정보를 제공합니다.

당사는 다음과 같이 **명시적으로** 약속합니다:
- 사용자 프로파일링이나 행동 분석은 수행되지 않습니다

### 4.2 데이터 보안 기술 조치

1. **전송 계층 보안(TLS): Cloudflare에서 제공하는 **TLS 1.3** 프로토콜을 사용하여 전체 전송을 암호화하며, HSTS 및 인증서 투명성(CT) 로그가 활성화되어 있습니다.
2. **암호 암호화**: **Argon2id** 알고리즘을 사용하며, 메모리 매개변수는 64MB, 반복 횟수는 1회, 병렬 처리 수준은 2입니다. 각 암호는 독립적인 임의 솔트를 사용합니다.
3. **데이터베이스 암호화**: 사용자 이름과 이메일 주소가 저장된 테이블스페이스에 투명 데이터 암호화(TDE)가 활성화되어 있습니다.
4. **접근 제어**: 데이터베이스 접근 권한은 최소 권한 원칙을 따르며, **핵심 운영 및 유지 관리 담당자**만 접근 권한을 가집니다.

## 5. 제3자 데이터 공유 및 공개

### 5.1 Cloudflare와의 데이터 공유

당사는 웹사이트 접속 속도 및 보안 향상을 위해 Cloudflare의 글로벌 CDN 서비스를 이용합니다. Cloudflare는 **독립적인 데이터 관리자**로서 서비스 제공 과정에서 사용자의 IP 주소, 접속 시간, 브라우저 지문 및 기타 정보를 수집할 수 있습니다. 당사는 Cloudflare의 데이터 수집 방식을 통제할 수 없으며, Cloudflare의 데이터 처리 활동은 Cloudflare 자체의 [개인정보 보호정책](https://www.cloudflare.com/privacypolicy/)에 따릅니다.

### 5.2 Microsoft와의 데이터 공유

Microsoft 계정 로그인 또는 연결 기능을 사용하는 경우, 당사는 Microsoft로부터 사용자의 사용자 ID, 표시 이름, 아바타 이미지 및 기타 정보를 수집합니다. Microsoft의 데이터 처리 활동은 Microsoft 자체의 [개인정보 보호정책](https://www.microsoft.com/privacy/privacystatement)에 따릅니다.

### 5.3 국경 간 데이터 전송 설명

Cloudflare(미국 회사)의 CDN 서비스의 기술적 특성상 데이터 전송 경로는 중국 이외 지역에 위치한 서버 노드를 경유할 수 있습니다. 그러나 당사의 주요 개인정보 데이터베이스는 중국 본토에 위치한 서버에 저장되며, **해외 제3자에게 개인정보를 적극적으로 제공하지 않습니다.**

**보안 조치:**
1. **암호화된 전송:** 모든 데이터 전송은 TLS를 사용하여 암호화되므로 전송 중 데이터 도난이나 변조를 방지합니다.
2. **최소화 원칙:** 서비스 제공에 필요한 최소한의 데이터만 전송합니다.

## 6. 사용자 권한 및 행사 방법

### 6.1 접근권 (알 권리)

- **셀프 조회:** [계정 관리 페이지](/account/dashboard)에 로그인하여 사용자 이름, 이메일, 가입일, 아바타 URL 등 저장된 모든 정보를 확인할 수 있습니다.
- **데이터 내보내기:** 계정 관리 페이지의 "보안 설정" 영역에서 "데이터 내보내기" 버튼을 클릭하여 JSON 형식의 전체 데이터를 다운로드할 수 있습니다.

### 6.2 수정권

사용자 이름, 비밀번호, 아바타 URL은 계정 관리 페이지에서 **개별적으로** 수정할 수 있으며, 변경 사항은 즉시 적용됩니다.

### 6.3 삭제권 (잊힐 권리)

계정 관리 페이지에 로그인하여 "계정 삭제" 버튼을 클릭하고 이메일 인증 코드를 입력하면 시스템에서 즉시 삭제 프로세스를 시작합니다.

### 6.4 OAuth 권한 관리

- **권한 조회**: [계정 관리 페이지](/account/dashboard)에 로그인하면 "OAuth 권한 관리" 영역에서 승인된 모든 타사 애플리케이션 목록을 확인할 수 있습니다.
- **권한 취소**: 계정 관리 페이지에서 언제든지 타사 애플리케이션에 대한 권한을 취소할 수 있습니다. 권한을 취소하면 해당 애플리케이션은 더 이상 계정 정보에 접근할 수 없습니다.
